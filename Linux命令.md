# Linux命令

## ifconfig能得到哪些信息

ifconfig命令可以获取以下信息：

1. 网卡名称
2. 网卡MAC地址
3. IP地址
4. 子网掩码
5. 广播地址
6. 网络状态
7. 接收数据包数目
8. 发送数据包数目
9. 接收错误数目
10. 发送错误数目
11. 接收丢失数目
12. 发送丢失数目
13. 网卡类型
14. MTU值
15. 网卡的物理地址
16. 网卡的IP地址
17. 网卡的广播地址
18. 网卡的子网掩码
19. 网卡的状态
20. 网卡的速度
21. 网络连接类型
22. 网络连接状态
23. 网络连接速度
24. 网络连接质量
25. 网络连接信号强度

## vim编辑器常用的快捷键和使用方法

### 1. 移动光标

h：向左移动一个字符

j：向下移动一行

k：向上移动一行

l：向右移动一个字符

w：向右移动一个单词

b：向左移动一个单词

0：移动到行首

$：移动到行尾

gg：移动到文件开头

G：移动到文件结尾

### 2. 插入模式

i：在当前光标位置插入

a：在当前光标后插入

o：在当前行下方插入一行

O：在当前行上方插入一行

### 3. 删除文本

x：删除当前光标所在的字符

dd：删除当前行

D：删除从光标位置到行尾的所有字符

### 4. 撤销和重做

u：撤销上一次操作

Ctrl + r：重做上一次撤销的操作

### 5. 复制和粘贴

yy：复制当前行

p：在光标下方粘贴

P：在光标上方粘贴

### 6. 查找和替换

/：进入查找模式

n：查找下一个匹配项

N：查找上一个匹配项

:%s/old/new/g：全局替换 old 为 new

### 7. 保存和退出

:w：保存文件

:q：退出 Vim

:wq：保存并退出 Vim

ZZ：保存并退出 Vim

以上是 Vim 编辑器的一些常用快捷键和使用方法，希望对你有所帮助。
